Types of License

Slate pricing is dependent on what kind of user you are and how you wish to use the product:

    OpenBanking User

    As an Open Banking user you are able to connect using Slate Money, via our trusted intermediary, to any of your bank accounts for which Open Banking is supported.

    There are 2 modes of usage avaialable to OpenBanking users:


    Cloud Mode

    Your account and transaction data will be held by Slate in secure cloud storage. This enables you to enjoy the full benefits Slate across both the mobile and desktop apps.

      • Access to all your connected accounts, whether Open Banking or Offline, from anywhere via the mobile app
      • Your data is held by Slate in secure, encrypted cloud storage
      • Full mobile funcationality including alerts and notifications

    Local Mode

    This is also known as Total Privacy Mode. Your account and transaction data will be held by you in a file storage location of your choice. Only the Slate desktop app will have access to that data. The mobile app will have no visibility of the data and as such mobile access will be unavailable.

      • Access to all your connected accounts, whether OpenBanking or Offline, from the Slate desktop app only
      • Data access strictly from the desktop application.
      • User specified storage location
      • No mobile app connectivity, no mobile alerts

    Offline User

    We apprecaite that some users may not want to use OpenBanking. However, Slate still provides all the same analytical functionality in the case where transaction data sourced offline.

    Offline users are implicitly in Local Mode, meaning the same as above.

      • You have the responsibility to manually download data from your account holding bank/institution and upload to Slate desktop
      • None of the automation benefits of OpenBanking.
      • Data access strictly from the desktop application.
      • User specified storage location
      • No mobile app connectivity, no mobile alerts
